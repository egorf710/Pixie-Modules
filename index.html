Как подключить модуль?


1. Создайте Java проект

2. Подключите Pixie в свой проект


3. Создайте pixie файл.
Пример: main.pixie



4. Добавьте ваш модуль в папку modules.



5. В классе App.java введите:
          Pixie.addModule("ModuleName", new InputModule());
        
           //чтобы подключить модуль

Пример:
Pixie.addModule("Input", new InputModule());

            //подключаем модуль Input

6. В том же классе App.java введите:
Pixie.execute("путь до файла main.pixie");

Пример:
Pixie.execute("C:/Projects/Java/PixieProject/src/pixie/main.pixie");




Готово!, Вы подключили модуль

Input

Модуль Input, позволяет осуществить консольный ввод.

Поддерживает ввод: Text(String), Num(Integer, Float), Bool(Bool)


Пример: 
          import Input

          var val = Input()
            
          print(val)
